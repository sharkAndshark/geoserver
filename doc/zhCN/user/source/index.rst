#####################
GeoServer 用户手册
#####################

　　GeoServer是一个使用Java编写的，允许用户分享、编辑地理空间数据的开源软件。它在设计时就考虑到了互操作性，其支持使用开放标准发布多数主流格式的空间数据。

　　本手册是使用GeoServer的全面指南，无论您是本软件的新手还是资深用户，希望都能有所收获。
